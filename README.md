# Dota 2 NPC Ability Library

A comprehensive collection of Dota 2 custom abilities and spell implementations in Lua for modding and custom game development.

## Contents

- **dota-2-lua-abilities**: Lua implementations of Dota 2 hero abilities
- **SpellLibrary-master**: Additional spell library and resources

## Installation

1. Clone this repository
2. Copy to `yourdotafolder/game/dota_addons`
3. Launch Dota 2
4. Run in console: `dota_launch_custom_game dota-2-lua-abilities dota`

Note: Enable cheat mode for easier ability testing.

## Features

- Custom hero abilities in Lua
- NPC ability data files
- Particle effects and materials
- Custom UI elements
- Spell icons and resources
